# Contributing to Simple Interest Calculator

We welcome contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your changes.
3. Make your edits.
4. Open a pull request.

## Reporting Bugs
If you find a bug, please open an issue and include:
- A clear description of the bug.
- Steps to reproduce the bug.
- Expected behavior vs actual behavior.

## Suggesting Enhancements and Ideas
Please open an issue to discuss any new features, enhancements, or ideas you have!
